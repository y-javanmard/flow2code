## Quick Start
1. Clone the repository:
   ```bash
   git clone https://github.com/y-javanmard/flow2code.git
   cd flow2code
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
